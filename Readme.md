# 준비사항

이 프로그램을 사용하기 위해서는 몇 가지 준비사항이 필요합니다.

1. 지금 이거 다운로드
2. Python 설치
3. Selenium install
4. Chrome 드라이버 설정
5. login.txt 파일 수정

- 이 프로젝트 파일 다운로드

    ```

    ```
